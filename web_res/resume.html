shmeeload webserver

As mentioned on the homepage, this website is served by a custom webserver developed 100% from scratch using Golang and Typescript. This server was initially developed by my colleague, Alex Johnson, (check out his website here!) however we have decided to collaborate in order to grow and expand the project with hopes of creating a fun, feature rich, robust, and easy to use software package! I have already made contributions based on my experience with computer graphics and web-based rendering by implementing support for the efficient serving of 3D model/mesh/geometry formats such as .obj, .gltf, and .dae! You can see this feature put to use by the various 3D real-time renderings across my website which I implemented with threejs/WebGL.

For more information and source code, please check out the repo!.

rusty tracer

A CPU ray-tracer written in Rust featuring implicit geometric ray-object intersection algorithms and a Phong illumination model. This is a personal project I undertook to learn more about modern functional programming languages, specifically Rust, while also indulging in my passions of computer garphics, rendering, and light physics! I was also driven to work on this after I had completed Peter Shirley's ray tracing in one weekend book as I wanted to challenge myself and reinforce what I had learned about ray tracing!

For more information and source code, please check out the repo!.
